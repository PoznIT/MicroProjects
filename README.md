# MicroProjects

A collection of small, focused utility tools.

## Projects

### [TimePunch](TimePunch/)
A lightweight time tracking app for logging work hours. Features a simple punch-in/punch-out interface with CSV export.

To use, open `TimePunch/index.html` in a browser. Click on it in your file explorer, if it does not open in your browser, select `open with` and choose a browser in the list.
